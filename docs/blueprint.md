# **App Name**: Ace Interview

## Core Features:

- Selectable Roles: Allow users to select roles for mock interviews from a list.
- AI-Generated Questions: Generate interview questions based on the selected role using AI. This will involve an LLM that can intelligently reason like a 'tool' about when/if to incorporate different kinds of knowledge.
- Answer Recording: Record and submit interview answers.
- AI Feedback: Provide instant AI feedback and performance insights after each answer.
- Encouraging Theme: Show encouraging, clean user interface during mock interviews.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust. 
- Background color: Light blue-gray (#E8EAF6) with approximately 15% saturation to provide a clean, calming backdrop. 
- Accent color: Soft green (#8BC34A) for a sense of encouragement and growth; visibly distinct through higher brightness and saturation compared to the other hues. 
- Headline font: 'Space Grotesk' (sans-serif) for a tech-forward, contemporary look for headers. Body Font: 'Inter' (sans-serif) for longer sections of text.
- Use professional, clear icons to represent roles and feedback types.
- Subtle, positive animations to highlight correct answers and feedback points.
